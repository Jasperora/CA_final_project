# 2023 Fall NTUEE Computer Architecture Final Project: Single Cycle CPU with Cache

## Features

### CPU

Implement the following instructions:

- auipc, jal, jalr
- add, sub, and, xor
- addi, slli, slti, srai
- lw, sw
- mul
- beq, bge, blt, bne
- ecall

### Cache

- Write Back + Write Allocation + 2-way association
